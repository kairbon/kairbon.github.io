NginxеҸҚеҗ‘д»ЈзҗҶеҗҺпјҢServletеә”з”ЁйҖҡиҝҮrequest.getRemoteAddr()еҸ–еҲ°зҡ„IPжҳҜNginxзҡ„IPең°еқҖпјҢе№¶йқһе®ўжҲ·з«Ҝзңҹе®һIPпјҢйҖҡиҝҮrequest.getRequestURL()иҺ·еҸ–зҡ„еҹҹеҗҚгҖҒеҚҸи®®гҖҒз«ҜеҸЈйғҪжҳҜNginxи®ҝй—®Webеә”з”Ёж—¶зҡ„еҹҹеҗҚгҖҒеҚҸи®®гҖҒз«ҜеҸЈпјҢиҖҢйқһе®ўжҲ·з«ҜжөҸи§ҲеҷЁең°еқҖж ҸдёҠзҡ„зңҹе®һеҹҹеҗҚгҖҒеҚҸи®®гҖҒз«ҜеҸЈгҖӮ

Nginxзҡ„еҸҚеҗ‘д»ЈзҗҶе®һйҷ…дёҠжҳҜе®ўжҲ·з«Ҝе’Ңзңҹе®һзҡ„еә”з”ЁжңҚеҠЎеҷЁд№Ӣй—ҙзҡ„дёҖдёӘжЎҘжўҒпјҢе®ўжҲ·з«ҜпјҲдёҖиҲ¬жҳҜжөҸи§ҲеҷЁпјүи®ҝй—®NginxжңҚеҠЎеҷЁпјҢNginxеҶҚеҺ»и®ҝй—®Webеә”з”ЁжңҚеҠЎеҷЁгҖӮеҜ№дәҺWebеә”з”ЁжқҘиҜҙпјҢиҝҷж¬ЎHTTPиҜ·жұӮзҡ„е®ўжҲ·з«ҜжҳҜNginxиҖҢйқһзңҹе®һзҡ„е®ўжҲ·з«ҜжөҸи§ҲеҷЁпјҢеҰӮжһңдёҚеҒҡзү№ж®ҠеӨ„зҗҶзҡ„иҜқпјҢWebеә”з”ЁдјҡжҠҠNginxеҪ“дҪңиҜ·жұӮзҡ„е®ўжҲ·з«ҜпјҢиҺ·еҸ–еҲ°зҡ„е®ўжҲ·з«ҜдҝЎжҒҜе°ұжҳҜNginxзҡ„дёҖдәӣдҝЎжҒҜгҖӮ

и§ЈеҶіиҝҷдёӘй—®йўҳиҰҒд»ҺдёӨдёӘж–№йқўжқҘи§ЈеҶіпјҡ

  1. з”ұдәҺNginxжҳҜд»ЈзҗҶжңҚеҠЎеҷЁпјҢжүҖжңүе®ўжҲ·з«ҜиҜ·жұӮйғҪд»ҺNginxиҪ¬еҸ‘еҲ°TomcatпјҢеҰӮжһңNginxдёҚжҠҠе®ўжҲ·з«Ҝзңҹе®һIPгҖҒеҹҹеҗҚгҖҒеҚҸи®®гҖҒз«ҜеҸЈе‘ҠиҜүTomcatпјҢйӮЈд№ҲTomcatеә”з”ЁжҳҜж°ёиҝңдёҚдјҡзҹҘйҒ“иҝҷдәӣдҝЎжҒҜзҡ„пјҢжүҖд»ҘйңҖиҰҒNginxй…ҚзҪ®дёҖдәӣHTTP HeaderжқҘе°ҶиҝҷдәӣдҝЎжҒҜе‘ҠиҜүиў«д»ЈзҗҶзҡ„Tomcatпјӣ
  2. TomcatиҝҷдёҖз«ҜпјҢдёҚиғҪеҶҚеӮ»д№Һд№Һзҡ„иҺ·еҸ–зӣҙжҺҘе’Ңе®ғиҝһжҺҘзҡ„е®ўжҲ·з«ҜпјҲд№ҹе°ұжҳҜNginxпјүзҡ„дҝЎжҒҜпјҢиҖҢжҳҜиҰҒд»ҺNginxдј йҖ’иҝҮжқҘзҡ„HTTP HeaderдёӯиҺ·еҸ–е®ўжҲ·з«ҜдҝЎжҒҜгҖӮ

Nginx

еңЁд»ЈзҗҶзҡ„жҜҸдёӘlocationеӨ„ж·»еҠ д»ҘдёӢй…ҚзҪ®пјҡ

и§ЈйҮҠд»ҘдёӢдёҠйқўзҡ„й…ҚзҪ®пјҢд»ҘдёҠй…ҚзҪ®жҳҜеңЁNginxеҸҚеҗ‘д»ЈзҗҶзҡ„ж—¶еҖҷпјҢж·»еҠ дёҖдәӣиҜ·жұӮHeaderгҖӮ

  1. HostеҢ…еҗ«е®ўжҲ·з«Ҝзңҹе®һзҡ„еҹҹеҗҚе’Ңз«ҜеҸЈеҸ·пјӣ
  2. X-Forwarded-ProtoиЎЁзӨәе®ўжҲ·з«Ҝзңҹе®һзҡ„еҚҸи®®пјҲhttpиҝҳжҳҜhttpsпјүпјӣ
  3. X-Real-IPиЎЁзӨәе®ўжҲ·з«Ҝзңҹе®һзҡ„IPпјӣ
  4. X-Forwarded-ForиҝҷдёӘHeaderе’ҢX-Real-IPзұ»дјјпјҢдҪҶе®ғеңЁеӨҡеұӮд»ЈзҗҶж—¶дјҡеҢ…еҗ«зңҹе®һе®ўжҲ·з«ҜеҸҠдёӯй—ҙжҜҸдёӘд»ЈзҗҶжңҚеҠЎеҷЁзҡ„IPгҖӮ

й…ҚзҪ®еҲ°иҝҷдёҖжӯҘеҗҺпјҢиҝҳдёҚиғҪеҪ»еә•и§ЈеҶій—®йўҳгҖӮtomcatд№ҹйңҖиҰҒй…ҚзҪ®

еҰӮжһңдҪ еңЁзҪ‘дёҠжҗңзҙўвҖңJavaеҰӮдҪ•иҺ·еҸ–е®ўжҲ·з«Ҝзңҹе®һIPвҖқпјҢжҗңзҙўеҲ°зҡ„и§ЈеҶіж–№жЎҲеӨ§еӨҡжҳҜйҖҡиҝҮиҺ·еҸ–HTTPиҜ·жұӮеӨҙrequest.getHeader(вҖңX-Forwarded-ForвҖқ)жҲ–request.getHeader(вҖңX-Real-IPвҖқ)жқҘе®һзҺ°пјҢд№ҹе°ұжҳҜдёҠйқўеңЁNginxдёҠй…ҚзҪ®зҡ„HeaderпјҢиҝҷз§Қж–№жЎҲиҺ·еҸ–зҡ„з»“жһңзҡ„зЎ®жҳҜжӯЈзЎ®зҡ„пјҢдҪҶи§үеҫ—е№¶дёҚдјҳйӣ…гҖӮеӣ дёәж—ўз„¶Servlet APIжҸҗдҫӣдәҶrequest.getRemoteAddr()ж–№жі•иҺ·еҸ–е®ўжҲ·з«ҜIPпјҢйӮЈд№Ҳж— и®әжңүжІЎжңүз”ЁеҸҚеҗ‘д»ЈзҗҶеҜ№дәҺд»Јз Ғзј–еҶҷиҖ…жқҘиҜҙеә”иҜҘжҳҜйҖҸжҳҺзҡ„гҖӮдёӢйқўд»Ӣз»ҚдёҖз§ҚжӣҙеҠ дјҳйӣ…зҡ„ж–№ејҸгҖӮ

дҪҝз”ЁTomcatдҪңдёәеә”з”ЁжңҚеҠЎеҷЁпјҢеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®Tomcatзҡ„server.xmlж–Үд»¶пјҢеңЁHostе…ғзҙ еҶ…жңҖеҗҺеҠ е…ҘпјҡеҚіеҸҜ
